Internet & Its Applications

The Internet is a global network of interconnected computers. Applications include: Email Online banking Social media E-commerce Streaming services Cloud computing

WWW (World Wide Web)

The WWW is a system of interlinked web pages accessed via the Internet using browsers.

Domain Name

A Domain Name is the readable address of a website (e.g., www.example.com) that maps to an IP address.

HTTP-HyperText Transfer Protocol

HTTP is the protocol used for transferring web pages from servers to browsers.
